# PyStruct

> Interactive tutorials for algorithms and data structures — explained clearly, implemented in Python.

**Live site:** _coming soon (deploy to Vercel)_  
**Stack:** Next.js 15 · TypeScript · Tailwind CSS · Shiki

## Topics covered

| Category | Topics |
|---|---|
| Sorting | Bubble Sort, Merge Sort, Quick Sort |
| Searching | Binary Search, Linear Search |
| Data Structures | Linked List, Stack, Binary Search Tree |
| Graphs | BFS, DFS |

## Getting started

```bash
git clone https://github.com/poewer/pystruct.git
cd pystruct
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Contributing

Contributions are welcome! To add a new topic:

1. Open `src/data/topics.ts`
2. Add a new entry to the `topics` array following the existing schema
3. Open a pull request
